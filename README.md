### Описание
Данный проект представляет собой сайт онлайн-магазина, написанный на React.

### Демо   
https://martynenkoAl.github.io/react-shop

### Установка и запуск проекта   
1. git clone https://github.com/martynenkoAl/react-shop.git
2. npm i
3. npm run start

### Технологии
* React
* HTML5
* CSS3
* materializecss
* JavaScript
